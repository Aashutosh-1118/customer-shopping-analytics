# Customer Shopping Analytics

## 📌 Project Overview

An end-to-end data analysis pipeline exploring customer shopping behavior across 3,900 transaction records. The project covers data cleaning, feature engineering, SQL-based business analysis, and visualization — answering 10 real business questions about revenue drivers, customer segmentation, and subscription value.

## 🔍 Key Findings

- Subscribers and non-subscribers spend almost identically (59.49 vs 59.87 average) and have nearly the same repeat-purchase rate (~91% vs ~88%) — suggesting the subscription program isn't currently driving differentiated customer value.
- Male customers generated more than double the revenue of female customers (157,890 vs 75,191).
- ~80% of the customer base falls into the "Loyal" segment (3,116 of ~3,900 customers), indicating a mature, retention-driven customer base rather than one reliant on new acquisition.

*Full analysis, SQL queries, and visualizations in [`customer_behavior_analysis.ipynb`](./customer_behavior_analysis.ipynb).*

## 🧹 Data Cleaning & Feature Engineering

- Imputed missing `review_rating` values using the median rating per product category (rather than dropping rows or using a global median).
- Standardized column names to snake_case for consistency and readability.
- Engineered `age_group` (Young Adult / Adult / Middle-aged / Senior) via quartile binning on customer age.
- Engineered `purchase_frequency_days` by mapping qualitative purchase frequency labels (e.g., "Weekly", "Quarterly") to numeric day counts.
- Identified and removed `promo_code_used` as a fully redundant column (verified identical to `discount_applied` across all 3,900 rows before dropping).

## 📡 Pipeline

Cleaned data is pushed from pandas into a MySQL database using SQLAlchemy, then queried back into pandas for analysis and visualization — keeping the workflow reproducible end-to-end within a single notebook.

## 📊 Business Questions Answered

1. Total revenue generated by male vs. female customers
2. Customers who used a discount but still spent above the average purchase amount
3. Top 5 products by average review rating
4. Average purchase amount: Standard vs. Express shipping
5. Average spend and total revenue: subscribers vs. non-subscribers
6. Top 5 products by discount usage rate
7. Customer segmentation (New / Returning / Loyal) by purchase history
8. Top 3 most-purchased products within each category
9. Subscription rate among repeat buyers (>5 previous purchases)
10. Revenue contribution by age group

Each question is answered via a dedicated SQL query (CTEs and window functions used for segmentation and ranking), with results visualized using Matplotlib/Seaborn.
